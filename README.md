# Local BitNet Assistant

## Schritte

1. Modell herunterladen (siehe download-model.ps1)
2. Container bauen:
   powershell -ExecutionPolicy Bypass -File .\build.ps1
3. Container starten:
   powershell -ExecutionPolicy Bypass -File .\run.ps1

## VS Code

Continue Extension installieren (.vsix offline möglich)
config.yaml unter ~/.continue/ kopieren

## Offline

Nach Setup vollständig offline nutzbar.
